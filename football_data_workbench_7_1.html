FotMob Stats avancées par joueur (Player Lookup)

Pour chaque joueur FotMob individuellement : Passe, Attaque, Défense, Sortie de balle, ratings, etc. FotMob bloque les requêtes cross-origin et impose un anti-bot — on contourne en lisant les données directement depuis la page via un bookmarklet.

Étape 1 — Installer le bookmarklet (une fois)

  1. Affiche la barre de favoris du navigateur (Ctrl/Cmd + Shift + B)
  2. Glisse-dépose ce bouton dans la barre :
⚽ FotMob → Workbench
Astuce: tu peux aussi clic droit sur le bouton → "Ajouter aux favoris".

Étape 2 — Capturer un joueur

  1. Va sur la page d'un joueur, ex: fotmob.com/players/12994/lionel-messi
  2. Attends que la page soit chargée
  3. Clique sur le bookmarklet "⚽ FotMob → Workbench"
  4. Reviens ici et colle (Ctrl/Cmd + V) dans le champ ci-dessous

Joueurs capturés (0)

Accès programmatique : Ouvre la console (F12) → tape window.data pour voir tous les datasets chargés. window.data.player_current contient le dernier joueur capturé. Tu peux faire des analyses directement en JS, ou copier vers ton notebook Python (via copy(window.data) dans la console Chrome).

FBref Capture par bookmarklet (joueurs & équipes)

FBref bloque les requêtes cross-origin. Même principe que FotMob : un bookmarklet lit les données directement depuis la page ouverte dans ton navigateur, puis les colle ici. Deux bookmarklets disponibles : joueur et équipe.

Étape 1 — Installer les bookmarklets (une fois)

  1. Affiche la barre de favoris (Ctrl/Cmd + Shift + B)
  2. Glisse chaque bouton dans la barre de favoris :
📊 FBref Joueur → Workbench 🏟️ FBref Équipe → Workbench
Astuce : clic droit → "Ajouter aux favoris" si le glisser-déposer ne fonctionne pas.

Étape 2 — Capturer

Joueur : va sur fbref.com/en/players/[id]/[nom], attends le chargement, clique "📊 FBref Joueur → Workbench".

Équipe : va sur fbref.com/en/squads/[id]/[equipe], clique "🏟️ FBref Équipe → Workbench".

Reviens ici et colle (Ctrl/Cmd + V) ci-dessous.

🎯 Moteur de recommandation joueur → équipe · basé sur Understat

Charge d'abord les données Understat : soit ⚡ Understat Live (données fraîches, recommandé) soit Understat saison 2025/26 via GitHub. Le moteur normalise les stats par poste (percentiles), construit le profil de l'équipe cible, puis calcule la similarité cosinus entre chaque joueur et le trou à combler.

Accès programmatique : Ouvre la console (F12) → tape window.data pour voir tous les datasets chargés. window.data.player_current contient le dernier joueur FotMob capturé. window.data._fbref_last contient la dernière capture FBref.
